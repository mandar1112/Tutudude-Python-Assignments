# Tutudude-Python-Assignments

This repository contains assignments as the part of the **Python Programming Course on Tutedude**.

Each assignment is organized in a separate folders with a dedicated 'README.md'.
All tasks related to each assignment is completed are implemented and included in their respective folders.


## Course Overview
  - **Platform:** Tutedude
  - **Language:** Python


## Assignment Overview
  - **"Assignment"** | **"Topic Covered"** 

  - Assignment 1 | Basic Python Concepts, Operators, Input/Output
  - Assignment 2 | Control Structure, Conditional Statements, Loops
  - Assignment 3 | Functions, Recursion, Python Modules(Math Module)
  - Assignment 4 | File Handling, Exceptions, Error Handling
  - Assignment 5 | Dictionaries, Lists, List Slicing, Strings

### Note: If a task required an input file(e.g., sample.txt), create the file and import required modules before running the program.

## How to Run the Programs (PowerShell)
  ### Clone the repostiory
  git clone <repo-link>
  
  ### Navigate to an assignment folder
  cd Assignment
  
  ### Run the python file
  python filename.py
